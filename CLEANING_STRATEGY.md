# 保守型清洗策略 (Conservative Cleaning Strategy)

## 设计理念

**数据保真度 (Fidelity) > 阅读流畅度 (Readability)**

本项目采用"微创清洗"策略，专为多人协作构建数据集而设计。核心原则是：**不动骨头，只修皮毛**。

## 策略对比

| 维度 | 传统重写策略 | 保守型清洗策略 ✅ |
|------|------------|-----------------|
| **目标** | 提升阅读体验 | 保留原始信息 |
| **处理方式** | 意译、总结、优化 | 仅删除噪音 |
| **语言风格** | 转为书面语 | 保留口语风格 |
| **专业术语** | 可能被替换 | 完全保留 |
| **金句观点** | 可能被改写 | 一字不改 |
| **适用场景** | 个人阅读 | 数据集构建、多人协作 |

## 清洗规则

### ✅ 允许的操作

1. **删除口水词**
   - 无意义的口癖：嗯、啊、这个、那个、对吧、是不是
   - 重复词语：我我我、然后然后
   - 停顿填充词：就是说、怎么说呢

2. **修正识别错误**
   - 同音字错误（根据上下文判断）
   - 明显的 ASR 识别错误

3. **修复标点符号**
   - 添加合适的逗号、句号
   - 符合中文书写规范

### ❌ 禁止的操作

1. **不要改写**
   - 不要用书面语替换口语表达
   - 不要优化句式结构
   - 不要调整语序

2. **不要总结**
   - 不要省略任何内容
   - 不要用概括性语言替代原话
   - 不要合并段落

3. **不要添加**
   - 不要添加原文没有的内容
   - 不要添加解释性文字
   - 不要添加标题（除非在双轨制输出中）

## 双轨制输出格式

为了满足不同需求，系统提供三种下载格式：

### 1. 双轨制版本 (推荐用于数据集)

```markdown
## 段落 1

**清洗版**:
今天我们讲讲职场的事。首先声明，我没有卖职场课的打算。最近跟强哥沟通，我突然悟到了一个断言：职场的一切问题，本质上都是生态位问题。

<details>
<summary>📝 查看原文</summary>

今天我们讲讲职场的事,首先先跟大家说一下,我没有任何要做职场课,卖职场课的打算...只不过是呢,我最近在对于职场这个事情的话...我突然间悟了一些新的事情...叫做职场的一切问题,本质上都是生态位问题。

</details>

---
```

**优势**：
- ✅ 容错率高：AI 清洗错了可以查看原文
- ✅ 搜索精准：同时匹配清洗版和原文
- ✅ 引用方便：段落编号作为讨论锚点
- ✅ 可追溯性：保留完整的原始数据

### 2. 仅清洗版

纯净的清洗后文本，适合快速阅读和分享。

### 3. 仅原文

未经处理的原始转写文本，适合需要完全原始数据的场景。

## DeepSeek Prompt 设计

```markdown
# Role
你是一位专业的会议速记整理员。你的任务是对原始录音转写稿进行"微创清洗"。

# Principles (关键原则)
1. **最高保真度**：严禁通过意译、总结或概括来修改讲者的原话。
2. **仅删除噪音**：只删除口癖、识别错误、重复词语。
3. **不要改写**：不要用书面语替换口语表达。
4. **保留金句**：对于重要的定义、观点、金句，不要改动任何一个字。
```

## 使用场景

### 适合的场景 ✅

- 多人协作构建数据集
- 需要保留讲者原话的场景
- 学术研究、访谈记录
- 需要引用原文的文档
- 法律、医疗等对准确性要求高的领域

### 不适合的场景 ❌

- 个人笔记整理（可以用传统重写）
- 需要高度精炼的摘要
- 面向大众的出版物

## 技术实现

- **AI 模型**: DeepSeek Chat
- **Temperature**: 0.2（更保守的输出）
- **策略**: Surgical Cleaning（微创清洗）
- **输出格式**: Markdown with collapsible sections

## 示例对比

### 输入（原始转写）
```
今天我们讲讲职场的事,首先先跟大家说一下,我没有任何要做职场课,卖职场课的打算...只不过是呢,我最近在对于职场这个事情的话...我突然间悟了一些新的事情...
```

### 传统重写策略（不推荐）
```
今天分享职场话题。首先声明，我不打算开设职场课程。最近我对职场有了新的认识。
```
❌ 问题：丢失了"跟强哥沟通"等细节，改变了语气

### 保守型清洗（推荐）✅
```
今天我们讲讲职场的事。首先声明，我没有卖职场课的打算。最近跟强哥沟通，我突然悟到了一些新的事情。
```
✅ 优势：保留了所有关键信息和讲者语气

## 总结

保守型清洗策略的核心是：**让数据说话，而不是让 AI 说话**。

我们相信，在数据集构建和多人协作场景中，原始数据的完整性比阅读的流畅性更重要。
